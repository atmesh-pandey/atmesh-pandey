## 🏥 Featured Project — Doctor Appointment Booking System

> A full microservices-based healthcare platform built with Java, Spring Boot, Kafka, and Stripe.

| Service | Responsibility |
|---|---|
| 🏥 Doctor Service | Doctor profiles, specializations, availability slots |
| 👤 Patient Service | Patient registration, login, profile management |
| 📅 Appointment Service | Booking logic, slot management, rescheduling |
| 💳 Payment Service | Stripe payment gateway integration, webhook handling |
| 🔔 Notification Service | Kafka consumer — email/SMS alerts on booking & payment |
| 🌐 API Gateway | Single entry point, JWT validation, request routing |

**Tech highlights:**
- `Apache Kafka` for async decoupled notification events
- `Stripe Webhooks` for reliable payment confirmation
- `Feign Client` for clean inter-service communication
- `Eureka` for dynamic service discovery
- `Zipkin` for distributed request tracing
- `Spring Security + JWT` for stateless authentication
